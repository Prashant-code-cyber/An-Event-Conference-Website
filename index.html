About Conf-Pro

Confpro is a modern free event website template which you can use for conferences, forums, meetings and other gatherings. It is visually appealing, responsive, mobile-ready and optimized for fantastic performance and search engines. Confpro features a beautiful full-screen slideshow, parallax effect, countdown timer and sticky navigation.
